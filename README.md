# Abstract

This document establishes the official digital identity of the author by linking traditional academic identifiers, decentralized identity systems (DID), blockchain addresses, and cryptographic credentials into a single verifiable trust chain.

Its objectives are:

- establish scientific authorship;
- provide evidence of prior art;
- facilitate cryptographic verification;
- strengthen Digital Sovereignty;
- support Decentralized Science (DeSci);
- create a persistent identity independent of any institution.

This document is intended to remain publicly accessible and permanently verifiable.

# Purpose

Scientific knowledge increasingly exists simultaneously across multiple ecosystems:

- Universities
- Research repositories
- Git repositories
- Blockchain networks

Because these environments are decentralized, identity must also become decentralized.

This manifesto defines the cryptographic and academic anchors that identify the same legal person across all these ecosystems.
